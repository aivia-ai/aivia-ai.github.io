Claude Projectsは、テーマごとに会話や資料をまとめる箱として考えると分かりやすいです。たとえば「AIVIA運営」「AI副業」「勉強計画」のように分けると、毎回同じ説明をしなくて済みます。

Memoryは、よく使う前提や好みを覚えてもらう機能です。文章は短めがいい、専門用語を減らしたい、表で比較したい。こうした好みを覚えてくれると、次の依頼が楽になります。

ただし、何でも入れれば良いわけではありません。個人情報、会社の機密、まだ公開できない資料は、入力してよいかを先に確認します。

長く続く作業では、AIに覚えてもらうほど便利になります。一方で、覚えた内容が古くなることもあります。定期的に前提を見直し、今の目的に合っているか確認することが大切です。

Claudeの記事では、長文整理、文章作成、Artifacts、Projectsなどの便利さだけでなく、どの作業に向くか、どこで人が確認するかを分けて見ると使いやすくなります。

ポイント

  1. Projectsは、テーマごとに資料や会話をまとめる箱として考えると分かりやすいです。
  2. Memoryは、好みや前提を覚えてもらう機能ですが、何を覚えたか確認しながら使います。
  3. 個人情報や社外秘の資料を入れる前に、扱ってよい情報かを確認します。
