Netlifyは、HTML、CSS、JavaScriptで作った静的サイトを公開できるサービスです。最初は無料公開で十分です。

流れはシンプルです。公開用フォルダを作る。中に `index.html` があるか見る。Netlify Dropへフォルダを入れる。発行されたURLで表示を確認する。

注意点は、手動公開ではNetlifyがビルドを代わりにやらないことです。`npm run build` が必要なサイトなら、先に自分のパソコンで完成版を作ります。

無料枠でも始められますが、更新回数、画像の量、アクセスが増えると使用量を見ます。最初から有料にするより、公開、計測、改善の順番で進める方が失敗しにくいです。

公開後は3つだけ確認します。ページと画像が開くか。本番URLがサイト設定に入っているか。Search Consoleへ `sitemap.xml` を送れるか。ここまでできれば、最初の公開としては十分です。

ポイント

  1. Netlify Dropは、完成した公開用フォルダをブラウザへドラッグして公開する方法です。
  2. 手動公開ではNetlify側でビルドが走らないため、先に自分のパソコンで `public` や `dist` を作っておきます。
  3. 無料公開で始め、読まれ始めてから独自ドメインや有料枠を考える流れが安全です。
